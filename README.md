## Webcam requriements ##
1. at least 720p resolution
2. located in the top center of the screen

## Running the demo ##
1. Download repo
```
git clone https://github.com/araddor1/ReadHelper.git
```
2. install requriements:
```
pip install -r requirements.txt
```
3. run main.py
```
python main.py
```
4.Enter needed information

5.Enjoy!
